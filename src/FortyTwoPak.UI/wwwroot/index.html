CLI Usage

          
Run from the command line without launching the GUI:

          
FortyTwoPak build   <sourceDir> <output.vpk> [options]
FortyTwoPak extract <archive.vpk> <outputDir> [--passphrase X]
FortyTwoPak info    <archive.vpk> [--passphrase X]
FortyTwoPak validate <archive.vpk> [--passphrase X]
FortyTwoPak convert <input.eix> <output.vpk> [options]

Options: --passphrase, --compression, --algorithm, --mangle, --author, --comment

        

        

          
Metin2 Integration

          
After building a .vpk, integrate it with your client/server:

          
    
            
  1. Copy VpkLoader.cpp/h into EterPack/
    2. 
            
  2. Modify CEterPackManager to use the VPK loader
    3. 
            
  3. Rebuild client with VS2022
    4. 
            
  4. For server, add VpkHandler.cpp/h to game source
    5. 
          

          
See the Metin2Integration/ folder for ready-to-use files.

        

        

          
Security

          
    
            
  • AES-256-GCM per-file encryption with unique nonces
    • 
            
  • PBKDF2-SHA512 key derivation with 200,000 iterations
    • 
            
  • HMAC-SHA256 whole-archive tamper detection
    • 
            
  • BLAKE3 per-file integrity verification
    • 
            
  • SHA256-based filename obfuscation
    • 
            
  • Use strong, unique passphrases per archive
    • 
          

        

        

          
Format

          
VPK Layout
+--------------------------+
| Header       512 bytes   |  "42PK" magic, version, config
+--------------------------+
| Data blocks              |  4096-aligned, LZ4/Zstd/Brotli, AES-256-GCM
+--------------------------+
| Entry table              |  File index (encrypted)
+--------------------------+
| HMAC-SHA256   32 bytes   |  Covers all preceding bytes
+--------------------------+
